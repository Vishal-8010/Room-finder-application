## 📱 Responsive Breakdown

```
320px (Small Phone)
├─ Logo badge only (36×36px)
├─ Hamburger menu
├─ Search below header
└─ Full-width menu drawer

480px (Large Phone)
├─ Logo + brand name
├─ Menu drawer
├─ Search below header
└─ Starting to show more text

768px (Tablet/Desktop)
├─ Desktop navigation visible
├─ Mobile menu hidden
├─ Search in header center
└─ User profile dropdown

1024px+ (Large Desktop)
└─ Full optimization
```
